이미지 업로드 방법
폴더 안의 파일을 전체 선택(Ctrl+A)해서 업로드하세요. 파일명 규칙에 따라 물건에 자동 매핑됩니다. 이미지 없이 다음 단계로 넘어갈 수 있습니다.
파일명 규칙
매장 사진 — Overview 슬라이드 좌측 배치
파일명에 _사진 또는 _photo 포함  예: 와우산로_94_사진1.jpg
도면 — Floor Plan 슬라이드 별도 생성 (없으면 건너뜀)
파일명에 _도면 또는 _floor 포함  예: 와우산로_94_도면.png
물건 매핑 — 파일명의 도로명으로 자동 매핑
예: 와우산로_94_사진1.jpg → 주소에 "와우산로 94" 포함된 물건에 연결
매핑 결과
물건 주소매장 사진도면
미매핑 파일
사용법 — ① 카카오맵(map.kakao.com) 캡처 이미지 업로드 → ② 오른쪽 목록에서 물건 번호 클릭 → ③ 지도 위 해당 위치 클릭하여 핀 배치 → ④ 지도 확정 버튼 클릭
지도 없이 건너뛰면 지도 슬라이드가 생략됩니다.
