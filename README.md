# C&W 물건제안서 자동생성

Cushman & Wakefield Korea — Retail Leasing Group 전용 물건제안서 자동생성 앱입니다.

## 사용 방법

### Step 1. 엑셀 업로드 & 물건 선택
- 물건 정보가 담긴 `.xlsx` 파일을 업로드합니다
- 데이터는 **4행부터** 시작해야 합니다
- 컬럼 순서: No. / 구분 / 주소 / 층수 / 전용(py) / 전용(㎡) / 보증금 / 월임대료 / 관리비 / 인도시기 / 상태 / 특이사항 / PPT포함(✓/✗)
- 업로드 후 제안할 물건을 체크박스로 선택합니다

### Step 2. 이미지 폴더 업로드
- 이미지 파일을 전체 선택(Ctrl+A)해서 업로드합니다
- **파일명 규칙**
  - 매장 사진: `연무장길_14_사진1.jpg` (파일명에 `_사진` 또는 `_photo` 포함)
  - 도면: `연무장길_14_도면.png` (파일명에 `_도면` 또는 `_floor` 포함)
- 도로명이 일치하는 물건에 자동 매핑됩니다
- 이미지 없이 다음 단계로 넘어갈 수 있습니다

### Step 3. 지도 & 핀 배치
- [map.kakao.com](https://map.kakao.com)에서 원하는 지역을 캡처합니다
- 캡처 이미지를 업로드합니다
- 오른쪽 물건 목록에서 번호를 클릭한 후 지도 위 해당 위치를 클릭해 핀을 배치합니다
- 모든 핀 배치 후 **지도 확정** 버튼을 클릭합니다
- 지도 없이 건너뛰면 지도 슬라이드가 생략됩니다

### Step 4. 제안서 생성
- 클라이언트 및 담당 어드바이저 입력
- **PPT 제안서 생성** 버튼 클릭
- 생성된 프롬프트를 복사해서 Claude 채팅창에 붙여넣기
- PPT 파일 다운로드

## 슬라이드 구성

1. 표지
2. 물건 위치 지도 (카카오맵 + 번호 핀) — 지도 확정 시
3. 물건 목록표
4번~. 물건별 Overview (매장사진 좌측 55% + 정보 우측 45%)
   - 도면 있으면 다음 슬라이드에 Floor Plan 추가
마지막. 연락처

## 기술 스택

- 순수 HTML/CSS/JavaScript (별도 설치 불필요)
- [SheetJS](https://sheetjs.com/) — 엑셀 파싱
- Canvas API — 지도 핀 배치
